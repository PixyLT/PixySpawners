**Stacking spawners plugin for Spigot 1.16**

## Features
- 💎 **Ability to stack spawners on right-click.**
- 🧾 **Ability to get/give spawners with command.**
- 😍 **Nice looking GUI.**
- 🎈 **No need to install any dependencies.**

## Compatibility
Currently tested with:
`✅ Spigot 1.16.2`
